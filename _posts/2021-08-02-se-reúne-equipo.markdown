Este lunes el equipo de transición de la Gobernadora electa sostuvo una reunión con integrantes del el Fideicomiso Carretero, según fue dado a conocer por Carlos Lascurain Ochoa encargado del área. 

Esto tiene como fin el conocer acciones que se llevan a cabo dentro del marco del proceso de entrega-recepción, destacando que se acordó que se diera una serie de reuniones con el fin de poder acceder a la información correspondiente para la entrada de la administración 2021-2027.

Dentro de las acciones que se dictaminaron fue el continuar con estas reuniones para así poder proseguir con transparencia el proceso de entrega-recepción entre el Gobierno Estatal saliente y la administración entrante.
